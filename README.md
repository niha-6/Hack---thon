# CivicSewa

A platform for citizens to report local issues, track complaints, and get guidance on ward office procedures.

Live site → [civicsewa.vercel.app](https://civicsewa.vercel.app)

---

## Features

- Submit complaints with photos and GPS location
- Track complaint status in real-time
- AI chatbot for civic guidance (documents, procedures)
- Ward office dashboard to manage complaints
- Official broadcast/announcement channel
- English and Nepali language support

## Tech Stack

- HTML / CSS / JavaScript
- Firebase (Auth, Firestore, Storage)
- Google Gemini API (chatbot)
- ImgBB (photo uploads)
- Vercel (hosting)

## Setup

```bash
git clone https://github.com/niha-6/Hack---thon.git
cd Hack---thon
```

Add your Firebase config to `js/core/firebase-config.js` and your Gemini API key to `js/citizen/chatbot.js`, then open `index.html` with Live Server.
